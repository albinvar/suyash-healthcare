# Assets Folder

This folder contains all static assets for the Suyash Health Care Centre website.

## Structure

```
assets/
├── images/          # Image files (logos, photos, icons)
│   ├── logo/       # Company logos
│   ├── hero/       # Hero/banner images
│   ├── products/   # Product photos
│   ├── machines/   # Equipment photos
│   ├── services/   # Service images
│   ├── team/       # Team member photos
│   ├── gallery/    # General gallery
│   └── icons/      # Icon files
├── videos/         # Video files (future)
├── documents/      # PDFs, brochures (future)
└── fonts/          # Custom fonts (if needed)
```

## Best Practices

1. **Organization:** Keep files organized in appropriate subdirectories
2. **Naming:** Use descriptive, lowercase names with hyphens
3. **Optimization:** Always optimize assets before uploading
4. **Version Control:** Keep originals in a separate backup location
5. **Licensing:** Ensure all assets have proper usage rights

## Asset Preparation Checklist

Before adding assets to the project:

- [ ] Images are optimized and compressed
- [ ] File names are descriptive and follow naming convention
- [ ] Multiple sizes provided for responsive design
- [ ] Alt text/descriptions prepared
- [ ] Licensing/usage rights verified
- [ ] Backup of originals saved elsewhere

## Tools Recommended

- **Image Optimization:** TinyPNG, Squoosh, ImageOptim
- **Image Editing:** Photoshop, GIMP, Figma
- **Vector Graphics:** Illustrator, Inkscape, Figma
- **Favicon Generation:** RealFaviconGenerator
